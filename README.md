  📊 Walmart Sales Analysis using Power BI

📌 Overview

This project presents a comprehensive analysis of Walmart’s sales data using Power BI. The goal is to transform raw transactional data into meaningful insights that help in understanding business performance, customer behavior, and sales trends.

🎯 Objective

Analyze sales performance across branches and cities

Identify top-performing product categories

Understand customer purchasing behavior

Discover time-based sales trends

📂Dataset Overview

| Column Name      | Description |
|-----------------|------------|
| invoice_id      | Unique ID for each transaction |
| Branch          | Walmart branch |
| City            | City of transaction |
| category        | Product category |
| unit_price2     | Price per unit |
| quantity        | Units sold |
| date            | Transaction date |
| time            | Transaction time |
| payment_method  | Payment method |
| rating          | Customer rating |
| profit_margin   | Profit margin (%) |
| transaction_amt | Total transaction amount |
| profit_amt      | Total profit |
